# Loader Collection

A collection of **13 custom loading screen animations** built using **pure HTML** with inline CSS and JavaScript.

Each loader is self-contained in a single HTML file and can be used directly in web projects or for learning and inspiration.

## ✨ Features
- Pure HTML (no external CSS or JS files)
- Lightweight and fast
- Easy to customize
- Beginner-friendly
- Works with GitHub Pages

## 🚀 Usage
1. Open any loader link from above  
2. Copy the HTML code  
3. Paste it into your project  
4. Customize colors, text, or animation speed as needed

## 🎓 Purpose
This project is created as a **frontend practice showcase** and can be used in portfolios, demos, or learning projects.
